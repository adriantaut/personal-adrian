# Personal Assistant - Adrian Taut

Acesta este spațiul personal al lui Adrian, separat de muncă (zar-adrian).

## Limbă

Comunică în **română** by default. Switch to English only if explicitly requested.

## Domenii

- **SRL (DevOps Box SRL)** - administrare firmă, contabilitate, beneficii angajat, fiscalitate
- **Finanțe personale** - investiții, economii, planificare
- **Viață personală** - organizare, reminder-uri, proiecte personale
- **Achiziții & Research** - comparații produse, decizii

## Structura Repository

```
personal-adrian/
├── CLAUDE.md           # Acest fișier
├── profile/
│   └── context.md      # Context personal Adrian
├── srl/
│   └── beneficii.md    # Tracking beneficii angajat
├── notes/
│   └── daily/          # Note zilnice
├── finance/            # Finanțe personale
└── projects/           # Proiecte personale
```

## Convenții

- **Fișiere în română** pentru chestii locale (SRL, admin)
- **Date format:** YYYY-MM-DD
- **Git auto-sync:** Pull la start, commit+push la stop
- **Note zilnice:** `notes/daily/YYYY-MM-DD.md`

## Reminder-uri Recurente

Verifică `srl/beneficii.md` pentru:
- Prime/tichete de sărbători (Paște, Crăciun)
- Raportare zile lucrate pentru tichete masă

## Principiu

La fel ca la muncă: **"Nothing is written by hand. Everything is automated."**
